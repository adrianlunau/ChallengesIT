Challenge n° 2

En consola se pueden utilizar las funciones: getUser() y getAlbums()

  • getUser( callback ):

    Recibe como argumento una función callback.

  • getAlbums( id, callback ):

    Recibe como primer argumento un ID y como segundo argumento una función callback.

Callbacks:

  • - asyncAwait
  • - promises
  • - xmlHttpRequest
